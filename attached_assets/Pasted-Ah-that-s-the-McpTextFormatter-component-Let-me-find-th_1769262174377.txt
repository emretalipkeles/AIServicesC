Here's the key part - the scrollable table container:

// Render a parsed table as styled HTML
function renderTable(table: TableData, tableIndex: number): React.ReactNode {
  return (
    <div 
      key={`table-${tableIndex}`} 
      className="my-5 overflow-hidden rounded-xl shadow-lg border border-teal-200/60 dark:border-teal-700/40 bg-gradient-to-b from-white to-slate-50/50 dark:from-slate-800 dark:to-slate-900"
    >
      {/* THIS IS THE KEY - scrollable container with max height */}
      <div className="max-h-[480px] overflow-y-auto overflow-x-auto">
        <table className="min-w-full">
          <thead className="bg-gradient-to-r from-teal-600 via-cyan-500 to-teal-600 sticky top-0">
            <tr>
              {table.headers.map((header, colIndex) => (
                <th className="px-4 py-3.5 text-xs font-bold text-white uppercase tracking-wider whitespace-nowrap">
                  {header}
                </th>
              ))}
            </tr>
          </thead>
          <tbody className="divide-y divide-slate-100">
            {table.rows.map((row, rowIndex) => (
              <tr className="hover:bg-teal-50/50 transition-colors">
                {row.map((cell, colIndex) => (
                  <td className="px-4 py-3 text-sm whitespace-nowrap">
                    {cell}
                  </td>
                ))}
              </tr>
            ))}
          </tbody>
        </table>
      </div>
    </div>
  );
}
The Magic Combination:
Outer container - overflow-hidden rounded-xl for nice borders that don't get cut off

Inner scrollable div - max-h-[480px] overflow-y-auto overflow-x-auto

max-h-[480px] - limits vertical height so tall tables don't explode the bubble
overflow-y-auto - vertical scroll when table is tall
overflow-x-auto - horizontal scroll when table is wide
Table - min-w-full ensures table takes full width

Cells - whitespace-nowrap prevents text wrapping, keeping columns clean

Sticky header - sticky top-0 on <thead> keeps headers visible when scrolling vertically
