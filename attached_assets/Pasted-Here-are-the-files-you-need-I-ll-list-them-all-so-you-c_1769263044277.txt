4. The TABLE SCROLL Magic (most important!)
From mcp-text-formatter.tsx lines 617-673:

function renderTable(table: TableData, tableIndex: number): React.ReactNode {
  return (
    // Outer container - clips everything, nice rounded borders
    <div className="my-5 overflow-hidden rounded-xl shadow-lg border border-teal-200/60">
      
      {/* INNER SCROLL CONTAINER - THIS IS THE KEY */}
      <div className="max-h-[480px] overflow-y-auto overflow-x-auto">
        
        <table className="min-w-full">
          <thead className="bg-gradient-to-r from-teal-600 via-cyan-500 to-teal-600 sticky top-0">
            <tr>
              {table.headers.map((header) => (
                <th className="px-4 py-3.5 text-xs font-bold text-white whitespace-nowrap">
                  {header}
                </th>
              ))}
            </tr>
          </thead>
          <tbody>
            {table.rows.map((row) => (
              <tr>
                {row.map((cell) => (
                  <td className="px-4 py-3 text-sm whitespace-nowrap">
                    {cell}
                  </td>
                ))}
              </tr>
            ))}
          </tbody>
        </table>
        
      </div>
    </div>
  );
}
The 4 CSS Keys That Make It Work:
max-w-[85%] on bubble - prevents bubble from stretching full width
overflow-hidden on outer container - clips content and keeps rounded borders clean
max-h-[480px] overflow-y-auto overflow-x-auto on inner div - independent scrolling
whitespace-nowrap on table cells - prevents text wrapping, keeps columns aligned
